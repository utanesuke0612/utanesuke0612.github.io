|命令           |意义           |例     |
|:--            |--:            |:--:   |
|`:q`           |退出           |       |
|`:wq`          |保存退出       |       |
|`i`            |insert         |       |
|`a`            |append         |       |
|`「Ctrl」f `   |next page      |       |
|`「Ctrl」b`    |previous page  |       |
|`:10`          |第10行         |       |
|`gg`           |第一行         |       |
|`G`            |最后行         |       |
|`x`            |删除一个文字   |       |
|`dd`           |删除一行       |       |
|`yy`           |复制1行        |       |
|`n yy`         |复制n行        |       |
|`p`            |paste next     |       |
|`P`            |paste previous |       |
|`u`            |undo           |       |
|`/検索文字列`  |检索           |       |
|`n`            |next find      |       |
|`N`            |previous find  |       |
|`:ns/old/new`  |n 行目の最初の old を new に置換して終了       |       |
|`:ns/old/new/g`|n 行目の全ての old を new に置換して終了       |       |
|`:%s/old/new/g`|ファイル全体の検索語句を置換する               |       |
|`:%s/old/new/gc`|置換の度に確認を求める                        |       |
