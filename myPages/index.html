  • 第1章 C#简介
    • .NET Framework
      • 是一个平台，类似于Java的JVM，理论上是可以跨平台的，只要这个平台安装了NET。
      • .NET 主要包含了一个庞大的代码库，可以在客户语言如C#，VB中通过面向对象编程来使用这些代码。
    • .NET 应用程序工作原理
      • 如何运行：
        • CIL：通用中间语言，第一阶段编译为这种语言，类似于Java的字节码。
        • JIT：Just-In-Time，它把CIL编译为专用于OS和，目标机器结构的本机代码。这样OS才能执行应用程序。
      • 程序集：
        • 编译应用程序时，所创建的CIL代码存储在一个程序集中。
      • 托管代码
        • 编译CIL ⇒(程序集) ⇒JIT编译 ⇒(本机代码 )⇒ CLR执行代码
        • 即CLR管理着应用程序，其方式是管理内存，处理安全性以及允许跨语言调试等。
        • C#中只能编写在托管环境下的代码
      • 垃圾回收
        • 托管代码最重要的一个功能是垃圾回收，执行垃圾回收的时间并不固定，可能一秒钟会进行数千次检查，也可能几秒钟检查一次。
    • C#概念以及与.NET的关系
      • C#从C和C++语言演化而来，它是为.NET框架设计的语言。
